# README: Petstore API Test Cases

This directory contains detailed, non-redundant, and comprehensive test cases for each module of the Swagger Petstore API. Each module's test cases are in a separate markdown file:

- [pet-test-cases.md](pet-test-cases.md): All test cases for the Pet module
- [store-test-cases.md](store-test-cases.md): All test cases for the Store module
- [user-test-cases.md](user-test-cases.md): All test cases for the User module

Each file covers:
- Basic (positive, negative, edge, validation)
- Integration
- Data Chaining
- System/Load
- Rate Limiting
- End-to-End (E2E)

Test case format: title, description, preconditions, test steps, expected result.

> For any additional modules or custom requirements, please specify.
